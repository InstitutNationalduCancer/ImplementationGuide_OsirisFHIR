Resource Profile: Genome Entity

Official URL: https://build.fhir.org/ig/arkhn/arkhn-ig-osiris/StructureDefinition/genome-entity Version: 2.1.0
Active as of 2023-04-06 Computable Name: GenomeEntity

The genome entity profile is used to indicate the genome reference used to build the Annotation.

Usage:

  • Refer to this Resource Profile:
  • Examples for this Resource Profile: MolecularSequence/fhir-osiris-genome-entity-1
