This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Treatment related profiles

Information about patient treatments

Oncologic Adverse Event

Adverse event related to a cancer-related treatment associated to a grade from MEDDRA terminology (ex: 10047700: "Vomiting").
Oncology Surgery

Oncology-related surgery with a description of the outcome using RECIST categorys.
Oncology-related Medication Statement

Medication statement related to a cancer cure.
Response Evaluation

Response evaluation to an oncology treatment from RECIST terminology.
Treatment

Description of a treatment plan related to an Oncology Patient.

Clinical data related profiles

Osiris clinical data

Analysis

Assessment of the technique(s) used to establish the diagnosis(es)
Biological Sample

Description of a biological sample coming from an oncology Patient, with its storage condition.
Biomarker

Biomarker Observation about a patient from BiomarkerCode ValueSet.
Cause Of Death

Observation that indicates the cause of death of an Oncology Patient.
Family Cancer History

History of cancer in an Onco Patient Family.
Oncology Patient

Oncology patient defined by Osiris.
Oncology Patient Related Pathology

Related pathology of oncology patient.
Performance Status

Oncology-patient performance status observation (see https://www.ncbi.nlm.nih.gov/books/NBK97482/ for more information).
Sequencing Analysis

Sequencing analysis about a biological sample coming from a TumorPathologyEvent. This profile shows general information about the analysis done, additional omics results can be found in SNP, Fusion, CopyNumber, Expression and Annotation profiles.
TNM

TNM Classification of malignant tumors used to indicate the tumor, node and metastasis state of a Tumor Pathology Event.
Tumor Pathology Event

Tumor Pathology Event (TPE) Condition describing a tumoral event: reference to the parent TPE, laterality, morphology.
Vital Status

Observation that indicates the last known Vital status of an Onco Patient.

Omics related profiles

Osiris omics data

3 Prime Molecular Sequence

3 prime molecular sequence is for referencing the 3' edge of a Fusion Observation.
5 Prime Molecular Sequence

5 prime molecular sequence is for referencing the 5' edge of a Fusion Observation.
Annotation Reference

The annotation reference profile indicates the reference genome / pfam domain / database used to generate the Annotation instance linked.
Gene Expression

Observation of a Gene Expression variant derived from a SequencingAnalysis. The Expression profile is used to indicate the actionability, the pathogenecity and the gene expression quantity of the gene of interest.
Gene Fusion

Observation of a Gene Fusion derived from a SequencingAnalysis. The Fusion profile indicates the pathogenicity, the actionability and if the fusion has occured in the reading frame.
Genome Entity

The genome entity profile is used to indicate the genome reference used to build the Annotation.
Sequencing Device

Description of a sequencing device used for a Sequencing Analysis.
Single Nucleotide Polymorphism

Observation of a Single Nucleotide Polymorphism (SNP) derived from a Sequencing Analysis.
Validation

Validation information about data from a Variant / Expression / Fusion / CopyNumber profile.
Variant Annotation

The Annotation profile allows to add information about a SNP, a fusion, an expression or a copy number event. It is generally used to compare variant from a reference genome. There can be multiple instance of Variant Annotation Resource for one instance of SNP, CopyNumber, Expression or Fusion Resource.
Variant Copy Number

Observation of a Copy Number variant derived from a SequencingAnalysis. The CopyNumber profile is used to indicate the pathogenicity, the actionability, the loss of heterozygosity, the cytogenetic location, the start - end positions, the reference sequence assembly, the variation code, the segment intensity, the number of copy found and the Copy Number segment size.

General information profiles

Osiris general information

Cancer Organization

Cancer organization to describe the provider center or origin center of the onco-patient.
Clinical Trial Subject

Subject that belongs to a oncology-related Clinical Trial.
Consent for Genetic Data Usage

The genetic consent resource indicates if a patient has given his authorization to share and use its genetics data.
Oncology Clinical Trial

Oncology clinical trial used to associate a Onco Patient to a Clinical Trial

Radiomics Image Filters

Filtering parameters used for the imaging analysis
Radiotherapy course Summary

A summary of a course of radiotherapy planned or delivered to a patient
Radiotherapy Phase

Summary of the radiotherapy phases planned or delivered to a patient
Radiotherapy Plan

Radiotherapy plans planned or delivered to a patient
Radiotherapy Volume Profile

A volume of the body treated by radiotherapy (plannes or delivered).
ROI segmentation

Description of the segmentation method used allowing the deffinition of a region of interest (ROI).
