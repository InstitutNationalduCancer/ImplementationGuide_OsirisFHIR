Resource Profile: Tumor Pathology Event

Official URL: https://build.fhir.org/ig/arkhn/arkhn-ig-osiris/StructureDefinition/tumor-pathology-event Version: 2.1.0
Draft as of 2024-05-15 Computable Name: TumorPathologyEvent

Tumor Pathology Event (TPE) Condition describing a tumoral event: reference to the parent TPE, laterality, morphology.

Usage:
