Resource Profile: Tumor Pathology Event

Official URL: https://ig-osiris.cancer.fr/ig/osiris/StructureDefinition/tumor-pathology-event Version: 1.1.0
Draft as of 2025-04-22 Computable Name: TumorPathologyEvent

Tumor Pathology Event (TPE) Condition describing a tumoral event: reference to the parent TPE, laterality, morphology.
